How to link two HTML pages
==========================
1. Normally a web app is composed of multiple pages where each page contains different information like: home page,contact page,login page etc.
2. Now for the customer or web site visitor it will be very difficult to remember URL of every page.
3. So it is the responsibility of a web developer to provide proper navigation between pages.
4. And to do this we use tag called as ANCHOR TAG.
